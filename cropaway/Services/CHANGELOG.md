# Changelog

All notable changes to cropaway will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Rectangle crop mode with adjustable dimensions
- Circle crop mode with adjustable radius
- Custom freehand mask drawing
- Keyframe animation system with interpolation
- Real-time video preview with crop overlay
- FFmpeg-based export with hardware acceleration
- VideoToolbox H.264/HEVC/ProRes encoding
- Alpha channel support (ProRes 4444)
- JSON metadata export for crop settings
- Professional playback controls (JKL shuttle)
- Frame-by-frame stepping
- Zoom controls for precise editing
- Undo/Redo support for crop operations
- Multiple video project management
- Drag-and-drop video import
- Comprehensive keyboard shortcuts
- Progress tracking for exports

### Technical
- SwiftUI-based UI
- AVFoundation for playback
- FFmpeg integration for export
- CoreGraphics for mask generation
- Combine for reactive updates
- Swift Concurrency (async/await)
